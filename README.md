# Django Backend

Django boilerplate with JWT Authentication

## Installation

### 1. Clone this repository  

```bash
git clone git@github.com:[org | username]/[repository].git
```

Or use this if you're using HTTPS

```bash
git clone https://github.com/[org | username]/[repository].git
```

#### 2. Setup environment

```bash
make setup
make install
```

#### 3. Run server on your computer

```bash
make run
```
